# Technical Manual — VW Gol GTI AB9 8V (1995)

🇬🇧 English

Interactive technical manual for the **Volkswagen Gol GTI 2.0 8V (AB9 body, 1995)**, AP 2000 engine (EA827), gasoline/ethanol — based on factory documentation and a reference car with all-original components.

**➡ Open the manual:** open `index.html` or visit the published GitHub Pages version.

## Contents

- **Electrical** — alarm, power windows and central locking (module and direct-wiring versions), instrument cluster, tachometer and trip computer, warning lights, pinouts and VW part numbers
- **Engine** — electronic fuel injection (static/dynamic fault codes), TFI ignition, fuel system, injectors, intake manifold, exhaust
- **Transmission** — gearbox (disassembly, shafts, synchronizers, ring and pinion, assembly), clutch, reinstallation torque specs
- **Reference** — full technical specs and mileage-based maintenance schedule

## Features

- Search by part, terminal or VW part number
- Light/dark theme
- Responsive layout (works on phones — ideal for garage use)
- Single self-contained file — works offline once loaded

## How to publish (GitHub Pages)

1. Push `index.html` and this `README.md` to the repository
2. In **Settings → Pages**, select the main branch and the `/ (root)` folder
3. The manual will be available at `https://YOUR-USER.github.io/REPO-NAME/`
